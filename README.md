# FluxGeo-Releases
FluxGeo official desktop releases (Windows, Linux) — a professional 2D geometry engine for exact constructions, function plotting, and narrative proof authoring.

FluxGeo 1.0 Beta — Initial Public Release

This is the first public beta release of FluxGeo.

FluxGeo is a professional 2D geometry environment featuring:

- exact constructions (CGAL-based)
- snapping-driven interaction
- function plotting (explicit and implicit)
- geometric transformations
- narrative proof authoring with HTML export

## Downloads

- Windows (Recommended): standard OpenGL version
- Windows (Software OpenGL): fallback for restricted environments
- Linux (x86_64): portable build

## Notes

- No installation required (portable)
- Keep all files in the extracted folder
- On Linux, run `./run-fluxgeo.sh`

## Known limitations (Beta)

- Some environments (VMs, sandbox) may not support OpenGL
- Performance may vary depending on GPU/driver
- Tablet/mobile web mode is still being refined
